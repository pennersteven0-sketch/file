# **App Name**: ConexPro

## Core Features:

- Job Scheduling: Schedule jobs with date, time, and team assignments.
- Quote Generation: Generate new quotes for potential jobs, including cost estimation based on materials and labor.
- Team Job View: Allow team members to view all scheduled jobs and their details.
- Job Details: Include comprehensive job details such as location, client information, and specific tasks. Optionally, AI parses the job description into the necessary task descriptions as a tool.
- Notifications: Real-time notifications for job updates and schedule changes for both the manager and the team members.
- Mobile Responsiveness: The Application is easily viewable on mobile devices and adjusts to different device sizes.
- Data Storage: Data about jobs, quotes, clients and other persistent information should be stored in a database.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5) to evoke trust, stability, and professionalism.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Orange (#FF9800) for CTAs and important notifications to draw attention.
- Font: 'Inter' sans-serif font, suitable for headlines and body text
- Use clear, modern icons from a set like Material Design Icons to represent actions and categories.
- Implement a clean and intuitive layout optimized for mobile screens, with easy navigation and clear information hierarchy.
- Use subtle transitions and animations for feedback and to guide the user through different actions within the app.
